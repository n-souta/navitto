# Navitto 製品・販売仕様

エディション、価格、今後の工程を定義する。詳細な機能仕様は `spec-plugin.md` / `spec-features.md` を参照。

---

## エディションと価格

### 無料版
- **価格:** ¥0（永久無料）
- **機能:**
  - 目次自動連携
  - H2選択機能
  - 表示開始位置制御（即時 / 選択した最初の見出しを通過後）
  - デザインプリセット（シンプル・テーマ準拠）
  - 横スクロール / 均等割

### 有料版（アイコン機能）
- **価格:** ¥1,000（買い切り）
- **機能:**
  - 無料版の全機能
  - アイコン選択機能（Font Awesome 同梱）
  - 見出しごとのアイコン設定
  - （画像自動連携・よく使うアイコン等は将来拡張の検討）

**現時点:** アイコン機能は実装済み。ライセンス検証未実装のため、有料版機能も無料で利用可能。ライセンス検証実装後に制限予定。

### 将来の Pro 版（公式サイト構築後）
- **価格:** 未定（¥3,000〜5,000 を検討）
- **機能（検討）:**
  - 無料版 + 有料版の全機能
  - アナリティクス
  - A/Bテスト
  - CTAボタン
  - カスタム画像設定
  - セクション背景色連動

## ライセンス検証の仕組み（予定）

1. ユーザーが Gumroad で購入
2. ライセンスキーを自動生成（Gumroad の機能または別途）
3. ユーザーが WordPress の「設定 → Navitto → ライセンス」でキーを入力
4. Gumroad API 等でキーを検証
5. 有効ならアイコン機能を利用可能に

**設定画面イメージ:**
```
設定 → Navitto → ライセンス
┌────────────────────────────┐
│ ライセンスキー              │
│ [___________________]      │
│ [有効化]                   │
│ ステータス: 未登録          │
└────────────────────────────┘
```
