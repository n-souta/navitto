WordPress.org 申請の準備 - 資材メモ
====================================

1. readme.txt
   - ルートの readme.txt を整備済み。
   - Contributors に WordPress.org ユーザー名を記入すること。
   - Stable tag は申請バージョン（1.2.0）と一致させること。

2. スクリーンショット
   - 1枚以上（推奨4枚）、推奨サイズ 1280x720。
   - SVN の assets フォルダに screenshot-1.png, screenshot-2.png ...
   - 内容目安: (1)固定ナビ表示例 (2)投稿編集メタボックス (3)カスタマイザー (4)設定画面
   - 設定画面を写す場合はライセンス欄は写さない（キー漏洩防止）。範囲を切るかぼかすこと。

3. プラグインアイコン
   - 128x128 または 256x256（PNG/JPG）。
   - プラグインの assets/ に icon-128x128.png を配置。WordPress.org 表示用に SVN の assets にも同ファイルを配置。
   - 未作成の場合は 128x128 の PNG を用意（ナビ／目次を連想するシンプルな図案を推奨）。

4. ライセンス・GPL
   - navitto.php ヘッダー・readme.txt・LICENSE で GPL v2 表記済み。対応済み。
